# 💬 Chat Application

## 📌 Описание
Этот проект представляет собой простое веб-приложение для обмена сообщениями между пользователями. Реализована регистрация, аутентификация и возможность отправки сообщений между зарегистрированными пользователями.

---

## 🚀 Функциональность
- ✅ **Регистрация пользователей**
- 🔐 **Аутентификация с использованием Spring Security**
- ✉️ **Отправка и получение сообщений**
- 🗄️ **Хранение пользователей и сообщений в базе данных Postrgresql**
- 🔑 **Хеширование паролей с использованием BCrypt**

---

## 🛠 Используемые технологии
- **🖥 Spring Boot** – основа приложения
- **🔐 Spring Security** – аутентификация и авторизация
- **🗄 Spring Data JPA** – работа с базой данных
- **💾 Postgresql Database** – база данных Postgresql
- **🌿 Thymeleaf** – шаблонизатор для фронтенда
- **📦 Lombok** – упрощение работы с сущностями

---

## ⚙️ Установка и запуск

### 1️⃣ Клонирование репозитория
```sh
 git clone https://github.com/ult1mq/ChatJava.git
 cd ChatJava
```

### 2️⃣ Запуск приложения
Приложение использует базу данных Postgresql и не требует дополнительной настройки.

```sh
 mvn spring-boot:run
```

После успешного запуска приложение будет доступно по адресу:
```
http://localhost:8080
```

---

### 3️⃣ Доступные эндпоинты
| 🔄 **Метод** | 🌐 **URL** | 📖 **Описание** |
|--------|-------------------|--------------------------|
| **GET** | `/login` | Страница входа |
| **GET** | `/register` | Страница регистрации |
| **POST** | `/register` | Регистрация нового пользователя |
| **GET** | `/chat` | Чат |
| **POST** | `/send` | Отправка сообщения |
| **POST** | `/logout` | Выход из системы |

---

**Параметры для входа:**
- 🛠 **JDBC URL:** `jdbc:postgresql://localhost:5432/&your_db`
- 👤 **Username:** `&your_username`
- 🔑 **Password:** `&your_password`

## 📂 Структура проекта
```bash

├── src/main/java/org/ult1mma/chatapplication/

```
